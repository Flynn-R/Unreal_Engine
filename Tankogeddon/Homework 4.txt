Задание 1
Пушки были добавлены с помощью дублирования класса BP_ProjectileCannon и изменения некоторых характеристик (материала для меша и урон снаряда), причём для каждой пушки были добавлены уникальные снаряды с помощью дублирования класса BP_Projectile.

Задание 2
Смена оружия была реализована с помощью привязки действия к клавише "пробел". При смене оружия проверяется наличие второго оружия. Если его нет, то ничего не происходит, если оно есть, то оно заменяется на текущее в методе ATankPawn::SetupCannon(TSubclassOf<ACannon>), сохраняется значение количества оставшихся боеприпасов и устанавливается новое. При подборе нового оружия заменяется только текущее, для него же обновляется счётчик снарядов. Счётчик снарядов для каждой пушки свой, что обеспечивается с помощью вспомогательной переменной ATankPawn::SecondaryAmmoLeft, куда сохраняется текущее значение количества снарядов на активном оружии сразу после нажатия клавиши "пробел".

Задание 3
Класс AAddAmmoBox является наследником класса AAmmoBox. С помощью переопределения виртуального метода OnMeshOverlapBegin(), внутри которого при коллизии игрока с объектом класса вызывается метод ACannon::SetAmmo() для текущего оружия игрока, устанавливающий новое значение количества снарядов. Для поддержки вызова метода ACannon::SetAmmo() в класс ATankPawn был добавлен метод ATankPawn::GetCannon().
